X射线荧光镀层检测技术发现隔离开关
触头镀银层厚度不合格
设备类别： 【隔离开关】
案例名称： 【X射线荧光镀层检测技术发现隔离开关触头镀银层厚度不合格】
单位名称： 【国网浙江省电力公司电力科学研究院】
技术类别： 【材质检测】
1. 案例经过
2010年1月17日～1月20日期间，对220kV荷花变工程开展基建阶段金属技术监督工作，对隔离开关触指镀银层厚度进行抽检。分别抽取了湖南长高、江苏如高、宁波阿鲁亚、温州昌泰等四个厂家的产品，采用X射线荧光法方法进行检测，发现湖南长高高压开关集团股份公司GW-2520W型、温州昌泰电气有限公司GW4-405D型等触头镀银层厚度均未达到DL/T486-2010《高压交流隔离开关和接地开关》第5.107.5规定的“导电杆和触头的镀银层厚度应≥20μm”要求。后用金相法对X射线荧光镀层测量的结果进行了复验，结果证明X射线荧光法检测结果准确无误。
2. 检测分析方法
采用X射线荧光镀层测量技术对隔离开关触头进行检测。在不合格产品中抽取了湖南长高高压开关集团股份公司生产的GW-2520W型隔离开关静触头制成金相样观测其镀银层并在显微镜下测量厚度。
3. 经验体会
3.1．闸刀触头镀银层厚度直接影响触头接触电阻大小。
3.2  由于闸刀触头镀银成本高，如不对闸刀触头镀银层进行监督，生产厂家对闸刀触头镀银层厚度要求远小于标准要求的20μm，甚至为了降低成本有厂家采用便宜的锡代替贵重的银。浙江公司自2010年将此项目纳入基建阶段监督的必检项目，共发现镀银层不合格闸刀418把，用锡代替银闸刀107把。
3.2．X射线荧光法检测闸刀触头镀银层厚度是一种无损的检测方法，经检测合格的触头仍然可以返回基建现场安装使用。
3.3 X射线荧光法检测闸刀触头时需到现场拆卸触头，除极个别型号的闸刀因触头和连杆是一体的，其余触头均可以拆卸。厂家常常为了规避检测而找藉口不允许拆除触头，对此应加大技术监督的执行力度。
