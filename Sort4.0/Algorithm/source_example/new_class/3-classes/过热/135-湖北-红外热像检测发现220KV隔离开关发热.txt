案例十七：
设备类别：【隔离开关+红外热像检测发现220KV隔离开关发热+17】
案例名称：【红外热像检测发现220KV隔离开关发热】
单位名称：【国网湖北省襄阳供电公司】
技术类别：【红外热像检测】
1、案例经过
湖北省电力公司襄阳供电公司东津变电站220KV母线侧隔离开关1998年生产、1999年投运，型号为GW16-220/1600-31.5。2013年6月在迎峰度夏前期，运维人员对设备进行红外成像检测时，发现某一隔离开关C相中间触头导电带处温度异常，表面最高温度达257 ℃ ，负荷电流700A。正常相36 ℃，负荷电流700A，环境温度26℃。停电后经检查，该隔离开关中间触头导电带与接线板接合处表面烧损，随即对受损导电部分进行了更换处理。
2、检测分析方法
在对隔离开关进行红外热成像检测时，发现其中一相导电部位温度分部异常。 并对该导电部位进行了精确测温，选择成像的角度、色度，拍下了清晰的图如图    1所示，并将此情况上报生产管理部门。
对图谱的认真分析后发现隔离开关C相中间触头部位温度异常，表面最高温度257℃，负荷电流700A。 正常相36℃，负荷电流700A，环境温度26℃。根据《交流高压电器在长期工作时的发热》（GB763-90)3.2条相关规定，用螺栓连接的导体接合部分铜和铝（镀层为镀银）在空气中最高允许温度为115℃，而实测温度达257℃；且根据《DL/T644-2008带电设备红外诊断技术应用导则》 中的公式计算，相对温差δT为95.7%； 依据电流致热型设备缺陷诊断判据，判断热像特征是以中间触头导电带连接处为中心的热像，热点明显； 故障特征为接触不良； 缺陷性质为危急缺陷。
将隔离开关停电后进行检查，发现隔离开关中间触头铜导电带已过热脆化，导电带与接线板接合处已烧损，螺栓烧损接近脱落如图2所示。随后对受损导电部分进行了更换处理，运行后红外热像检测合格。
3、经验体会
通过大量运行实践说明，红外热成像检测对发现设备过热缺陷是行之有效的方法。为防范导电接触部位发热，应做好以下几点：
（1）除了例行周期性红外测温以外，适当增加重要设备间隔的红外测温工作，及时发现、处理发热隐患，防止恶性发展。
（2）加强设备检修工艺和质量。导电接触面处理严格按工艺要求执行，可大幅减少发热隐患。
（3）对已发现处理的严重过热设备，举一反三，结合停电及时检查同期同类设备。
