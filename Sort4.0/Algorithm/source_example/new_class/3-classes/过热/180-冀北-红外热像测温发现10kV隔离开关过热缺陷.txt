设备类别：【隔离开关 案例 73】
案例名称：【红外热像测温发现10kV隔离开关过热缺陷】
单位名称：【国网冀北电力有限公司张家口供电公司】
技术类别：【红外热像测温】 
1. 案例经过
2013年3月20日，变电运维检测班在对姚家房站进行带电检测中发现10kV4号母线040-1B相隔离开关过热71.2℃，正常相9.5℃。
型号：GLKG-GW5-40.5DW，投运日期：2002年8月投运。
申请停电后，检修人员对其进行处理，发现隔离开关在运行中，刀闸触头部位存在连接不紧密点，随着污秽、雨水深入隔离开关导电接触面，氧化、腐蚀，造成隔离开关导电接触面接触日益劣化，最终导致隔离开关发热。随后对触头进行打磨，并重新调整刀闸后使设备恢复正常。随后的监测发现温度正常，缺陷消除。
2. 检测分析方法
10kV4号母线040-1B相隔离开关过热71.2℃，正常相9.5℃
3. 经验体会 
（1）应在迎峰度冬，迎峰度夏、保电等特殊期间对加强红外热像测温工作。
（2）检修时，应加强检修质量，防止由于检修不到位，不规范等原因引起的设备故障。

武兵星，国网冀北电力有限公司张家口供电公司

生产厂家：北京彼岸京都科技有限公司
