电网设备状态检测技术应用案例
设备类别： 【隔离闸刀】
案例名称： 【110kV云和变多处35kV设备发热】
单位名称： 【浙江省电力公司丽水供电公司】
技术类别： 【带电检测技术类别】
1、案例经过
2013年7月5日，检修公司（修试）对丽水地区的高负荷变电所云和变进行红外测温发现1#主变35kV主变闸刀A相和C相刀口、1#主变35kV母线闸刀A相刀口及1#主变35kV开关C相接头存在严重发现现象。在检测过程中发现1#主变35kV侧闸刀及开关设备存在温度异常，环境温度34.5℃，负荷电流550.34A，正常相温度52℃。经过停电检查发现闸刀刀口弹簧压接不良、刀口接触面氧化，开关接头接触不良。
2、检测分析方法
云和变在进行红外热成像检测时，发现1#主变35kV主变闸刀A相和C相刀口、1#主变35kV母线闸刀A相刀口及1#主变35kV开关C相接头温度存在异常。并对其进行了精确测温，选择成像的角度、色度，拍下了清晰的图谱，如图一、图二、图三、图四所示。
1）、1#主变35kV主变闸刀A相和C相刀口（图一、图二）温度异常分析：参照《带电设备红外诊断应用规范》DL/T664-1999附录A电流致热型设备缺陷诊断判据表A.1电流致热型设备缺陷诊断判据：设备类别和部位：隔离开关，刀口；热像特征：以刀口压接弹簧为中心的热像；故障特征：弹簧压接不良；此设备发热属于危急缺陷。
2）、1#主变35kV母线闸刀A相刀口（图三）分析：参照《带电设备红外诊断应用规范》DL/T664-1999附录A电流致热型设备缺陷诊断判据表A.1电流致热型设备缺陷诊断判据：设备类别和部位：隔离开关，刀口；热像特征：以刀口压接弹簧为中心的热像；故障特征：弹簧压接不良。此设备发热属于危急缺陷。
3）、1#主变35kV开关C相接头（图四）温度异常分析：参照《带电设备红外诊断应用规范》DL/T664-1999附录A电流致热型设备缺陷诊断判据表A.1电流致热型设备缺陷诊断判据：设备类别和部位：电器设备与金属部件的连接，接头和线夹；热像特征：以线夹和接头为中心的热像，热点明显；故障特征：接触不良。此设备发热属于一般缺陷。
变电检修人员根据红外判断的缺陷情况，对发热闸刀刀口及开关接头解体检查，发现闸刀刀口弹簧压接不良，接触面氧化，开关接头接触不良造成发热，与红外测温判断的缺陷情况基本一致。检修人员对闸刀刀口弹簧进行紧固，对刀口接触面打磨并涂凡士林，对开关接头进行了紧固。复役后，红外测温正常。
3. 经验体会
事实证明红外测温检查技术能够简单、快捷、有效的发现设备过热隐患，除周期性之外，特殊时期开展红外测温工作，能够及时快速的发现设备发热缺陷，尤其是迎峰度夏和迎峰度冬期间，加强红外测温检测，保证电网的安全运行。从多年来处理过的各类过热缺陷中发现导致接头过热的主要原因有以下几点：
1）、闸刀刀口、导线与金具螺丝连接处是设备的主要发热点，因为随着运行年限增加，接触面慢慢老化，容易导致发热，红外测温时要重点关注。
2）闸刀刀口接触面和金具接头接触面应涂凡士林，禁止涂导电膏，避免接触面接触不良导致过热。
3）导线与金具的连接采用压接形式，避免接触面与空气接触导致锈蚀、氧化。并尝试使用防止设备连接端子过热型紧固螺栓。
4）在刀闸检修工作过程中，针对平时运行负荷较大或者当地污染严重的
刀闸，重点关注动、 静触头的烧伤情况。
对以上设备检修工艺和质量提出了更高的要求，保证检修质量，全面考虑不留死角，而且在设备检修过程中对接头的处理将更具针对性，避免日后出现过热现象。
