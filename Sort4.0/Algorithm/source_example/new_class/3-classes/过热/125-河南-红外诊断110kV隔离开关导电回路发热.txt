设备类别： 【110kV隔离开关案例7】
案例名称： 【红外诊断110kV隔离开关导电回路发热】
单位名称： 【国网河南省电力公司濮阳供电公司】
技术类别： 【红外热像技术】
1.案例经过
2013年02月27日 ，国网河南省电力公司濮阳供电公司红外检测发现1号主变110kV主进线111甲隔离开关C相出线底座拐臂上部126℃，判定为危急缺陷。隔离开关型号：GW17A—126D1D2W, 2009年8月湖南长高集团股份公司制造。2009年10月投运后，一直对该类设备进行红外检测未发现异常。检查中发现拐臂内部软连接螺丝松动，局部烧熔。及时检修更换配件，运行后红外复测正常。
2. 检测分析方法
现场检测红外检测发现1号主变110kV主进线111甲隔离开关C相出线底座拐臂上部126℃，见图1-1、图1-2.从不同角度发现是隔离开关拐臂的内部发热，最高温度达到161℃。采用“表面温度判断法”，从远处水平拍摄，发现温度是从螺栓孔内看到的，初步判断是内部接头发热，见图1-3、图1-4。因为内部发热，看不清内部金属性质，所以，按照全面红外检测的辐射率0.91，未做调整。执行DL/ T664-2008电流致热型设备发热标准，电器设备与金属部件的连接部位表面温度≥130℃，或相对温差≥95%，判定为危急缺陷。检修人员检查中发现拐臂内部软连接螺丝松动，局部已有烧伤斑点。
解体检查：检修人员拆解隔离开关后，发现拐臂软连接螺丝紧固力度不足，表面氧化，局部烧损。见图图1-5、图1-6、图1-7。
缺陷原因：因为，封闭位置安装时受结构限制紧固螺丝不方便，未能全力矩紧固到位，新运行前几年由于负载小，接触部位温升低，发热点又处于封闭空间内部，散发到外部的热量消散，红外测温未能发现异常。随着系统负载不断增加，接触部位电阻增大、氧化严重使温度不断增加，高温点使空间温度无法正常散发，聚集形成高温区，红外热像热区明显，温差较高。
3. 经验体会
（1）隔离开关导电回路发热的红外诊断属于“电流致热型”，隔离开关的触头为典型的“可分合电接触”形式，而线夹、导杆等部位属于“固定电接触”，这些部位因暴露在外部，较容易发现发热问题。但此类隔离开关因为封闭设计，限制了早期发现问题的几率，红外检测中，应特别关注。因发热点处于底座包装的隐蔽处，多次在下部进行红外测温，造成温度不高的假象，只有在远处水平方向才能间接发现局部发热的高温点。所以，发现发热点后要从多角度检测，而内部实际温度要高于检测温度，高温点隐藏在内部的封闭状态，可能造成误判断，留下事故隐患。
（2）2012年4月发热点温度为35.4℃（见图1-8）到2013年3月发热点温度216.8℃。随着时间、负载的变化，发热点温度在一年时间内，由不到缺陷统计值的异常状态转变为危急缺陷。持续监督，即使低于缺陷标准也要做好记录，便于紧密红外监督变化，可控状态。
（3）电力工程师发现热点应考虑的问题：红外热像显示的是否热点的实际温度；发热点的发展趋势；考虑后续紧密监督与状态检修问题。
