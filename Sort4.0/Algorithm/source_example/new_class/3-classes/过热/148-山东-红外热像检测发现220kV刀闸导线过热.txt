案例十一
设备类别：刀闸
案例名称：红外热像检测发现220kV刀闸导线过热
单位名称：国网山东省电力公司菏泽供电公司
技术类别：带电检测技术
案例经过
220kVXX变电站1号主变220kV间隔2000年5月投运，导线型号为LGJ-400，运行状况良好。2013年8月23日进行红外带电检测。检测人员在使用红外热像仪检测过程中发现1主变220kV侧刀闸上引线导线温度异常（负荷电流222A，环境温度34℃）。导线局部温度过高，最高温为78.8℃，正常温度为43℃。
检测分析方法
导线局部发热，最高温处温度为78.8℃，正常点的温度为43℃，根据《DL/T644-2008带电设备红外诊断技术应用导则》中的公式计算，相对温差δT为83.2%，属于严重缺陷，初步判断为导线断股或锈蚀。
3. 处理情况：经停电检修发现，自刀闸侧线夹根部往外1.2米两端，外层两根导线存在压痕损伤，导线截面变细，有轻度氧化。经更换导线后缺陷排除。
4. 经验体会
红外热像仪可以有效防止电流致热、电压致热而造成的设备故障；同时还可通过热图像检测注油设备的油位。 
    a:电气装置：可发现接头松动或接触不良，不平衡负荷，过载，过热等隐患。这些隐患可能造成的潜在影响是产生电弧、短路、烧毁、起火。 
    b:变压器：可以发现的隐患有接头松动，套管过热，接触不良（抽头变换器），过载，三相负载不平衡，套管、储油柜油位，冷却管堵塞不畅等。空冷器件的绕组可直接用红外热像仪测量以查验过高的温度，任何热点都表明变压器绕组的损坏。其影响为产生电弧、短路、烧毁、起火。
c：电压互感器、避雷器等电压致热型设备的内部故障的检测。  
    d:各相之间的测量：检查电动机、充电机和其它设备的连线和连接器各相之间的温度是否相同。 
    e:不间断电源：确定UPS输出滤波器上连接线的发热点。一个温度低的点表明可能直流滤波线路是开路。
