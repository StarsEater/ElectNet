110kV鸣谦变电站35kV402断路器C相内部发热
设备类别：【断路器案例03】
案例名称：【红外热像带电检测110kV鸣谦变电站35kV 402断路器C相内部发热】
单位名称：【国网山西省电力公司晋中供电公司】
技术类别：【红外热像带电检测】
1、案例经过
2011年6月13日，国网山西省电力公司晋中供电公司检修试验专业在例行红外诊断中发现35kV 402断路器C相本体发热，以断路器下法兰为发热中心，热点明显。随后安排高压试验进行进一步验证。进行断路器回路电阻试验，证明C相断路器内部存在连接不良，致热原因为断路器动静触头连接接触不良导致的发热。对该设备进行解体，查看动静触头连接处，已严重烧烛，随即对动静触头进行更换。
2.检测分析方法
35kV402断路器为山东泰开电气公司LW8-35AG(CT)型产品，2008年7月26日投运。在红外诊断过程中发现402断路器C相本体异常，选择成像的角度，拍下表面最高温度82.4℃，正常温度34℃，环境参照体21℃，温差48.4K，相对温差78%（图1）。当时的负荷电流为406A。初步判断致热原因可能是因为断路器内部动静触头接触不良引起的发热。执行DL/T664-2008标准,表A.1电流致热型设备缺陷诊断判据,断路器故障相温度超过80℃，根据表面温度判断表和同类比较判断法,定性为电流致热型的危急缺陷。
立即要求该设备退出运行，安排高压试验进行进一步验证。进行回路电阻试验，发现一次绕组C相为356μΩ高于其他两相，证明断路器C相内部存在连接不良。高压试验结果与红外诊断结果一致，如表1所示。
安排检修人员对该设备进行解体，发现动静触头连接处，已严重烧烛，随即对动静触头进行更换。
检修人员处理后，再次对断路器进行回路电阻试验，发现一次绕组C相为78μΩ和其他两相无明显差别，证明断路器C相内部连接良好，如表2所示。
送电后24小时对402断路器进行红外复测，C相温度正常，与其他两相无明显差异。
3.经验体会
    红外诊断技术是一种诊断电气设备缺陷的先进测试技术，对及时发现、处理、预防重大事故的发生可以起到重大作用，具有远距离、不停电、不接触、不解体等特点,是电力系统主要的高压带电设备故障检测手段。在电力系统得到了广泛应用，具有良好的检测诊断效果。在国家电网公司发布的“输变电设备状态检修试验规程”中,列为电网设备的例行试验主要项目内容，在电力设备安全运行中起着至关重要的作用。所以电气设备的红外热成像检查，在很多方面可以代替传统的预防性维修工作。及时发现设备的缺陷和异常情况，为设备检修提供依据，为开展状态检修创造条件，提高设备运行的可靠率。
1、在日常巡视中，不仅要针对设备接头、接点进行测温。更要对电流、电压互感器、避雷器、变压器油枕、瓷瓶等设备进行仔细诊断，能够准确发现和处理一些容易忽略的设备缺陷。2、运用红外成像技术可以直观、高效地检查出设备内部发热，这是传统的检测单点温度的测温仪做不到的。3、应用多种检测手段综合判断设备性能，及时准确地诊断设备内部缺陷。
