设备类别：【隔离开关 案例 70】
案例名称：【红外热像检测发现500kV隔离开关过热缺陷】
单位名称：【国网冀北电力有限公司检修分公司】
技术类别：【红外热像测温】
案例经过
冀北电力检修分公司安各庄变电站500kV系统于2003年投入运行。2013年12月28日，运行中，对该变压器系统各电气设备外部各电气接头部位进行了红外热成像检测。在检测过程中发现：
王安一线5033-2刀闸C相动静触头连接处发热243度，A相52度，B相14度。
初步分析发热缺陷的原因是隔离开关在运行中，随着污秽、雨水深入隔离开关导电接触面，氧化、腐蚀，造成隔离开关导电接触面接触日益劣化，最终导致隔离开关发热。
检测分析方法
在例行的周期性红外测温过程中，发现该变电站500kV系统隔离开关存在着触头接触发热现象。对该发热点进行了精确测温，选择成像的角度、色度，拍下了清晰的图谱并将此情况上报运检部。
5033-2刀闸C相动静触头连接处发热主要原因是：
1、隔离开关各转动件卡涩,造成隔离开关小拐背角度板在合闸完成时未可靠超过180度（1-4mm)，使隔离开关夹紧力不足；
2、隔离开关六方杆长度不合适，使隔离开关下刀杆明显倾斜不垂直，造成隔离开关合闸到位以后，有两个同一侧动触杆未接触静触头（有极小的间隙），造成接触面接触不实。
现场用绝缘杆进行拉拽，是动静触头可靠接触，并加强监测，结合停电进行处理。
经验体会
大量事实说明红外热成像检测技术能够行之有效的发现设备过热缺陷，除了例行周期性红外测温以外，适当增加设备间隔停电前的红外测温工作，能够有效指导设备检修，提出大修、小修技改方案，提高检修工作效率和质量。我们发现导致连接处过热的主要原因有以下几点：
导线与隔离开关在运行中，随着污秽、雨水深入隔离开关导电接触面，氧化、腐蚀，造成隔离开关导电接触面接触日益劣化，最终导致隔离开关发热。
隔离开关在运行过程中，因为机构卡涩等原因造成刀闸合闸位置不正，导致隔离开关过热。
隔离开关在正常运行操作过程中触指多次经历引弧、燃弧、熄弧的损伤，致使产生过热。
隔离开采用材质金属部件强度低、质量差，外加运行工况较差，经过一段时间运行后机械性能大幅下降，导致产生过热。
