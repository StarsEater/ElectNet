设备类别： 断路器案例01
案例名称： 气体红外成像检漏技术发现110kVSF6 断路器漏气
单位名称： 国网青海省电力公司西宁供电公司
技术类别： 带电检测技术
1.案例经过
西宁供电公司变电运行人员在变电站巡视时，发现3台110千伏断路器气体压力指示偏低，断路器有漏气现象，专业技术人员利用红外检漏成像仪对有气体泄漏缺陷的断路器进行检测，发现3台110千伏SF6断路器漏点，即使停电处理后，断路器气体压力指示恢复正常。
2. 检测分析方法
专业人员利用红外成像检漏仪可清晰地看见泄漏出的SF6气体在空气中飘动，由于SF6气体在10-11μm波段比空气的辐射强，从红外图像上可看出泄漏点附近SF6气体浓度较大的地方较其他部分亮度高（红外图像上“ O ”形圈内），而且存在SF6气体飘动的影像，即图像上颜色较重的部分 。 
3.经验体会
红外成像检测SF6气体泄漏技术能迅速、准确对SF6电气设备的密封能力和状况进行定量检漏检测，确定泄漏源头，及时发现充气设备缺陷。
在状态检修工作中，检修人员可利用该项技术手段及时发现SF6气体绝缘设备漏气部位，从而极大的缩短检修中用于查找设备漏气部位的时间，提高设备缺陷处理的及时率和准确率，从而提高电网运行可靠性。
