案例十二：
设备类别：【断路器+电泄漏检测发现220kV断路器SF6气体泄漏+12】
案例名称：【带电泄漏检测发现220kV断路器SF6气体泄漏】
单位名称：【湖北省电力公司恩施供电公司检修分公司】
技术类别：【断路器带电检测】
1.案例经过
国网湖北省电力公司恩施供电公司某220kV变电站#1主变220kV侧断路器，型号LW53-252/Y3150-50，灭弧介质SF6气体，液压操动机构，新东北电气(沈阳)高压开关有限公司生产。SF6气体正常压力0.5MPa，报警压力0.45MPa，闭锁压力0.4MPa，生产日期2006年9月，于2007年9月投运，运行情况一直很稳定。2013年1月，运行人员发现该开关B相机构箱内SF6气体密度继电器压力指示下降比较快，日均压力下降0.0015MPa。检修人员于2013年1月进行了SF6气体泄漏检测，确定泄漏位置在B相机构箱内SF6气体截止阀与本体气室连接处，必须停电处理。2013月3月，联系厂家人员更换了SF6气体截止阀，并对开关本体进行回收SF6气体、抽真空、重新注入SF6气体，缺陷消除。
2. 检测分析方法
检修人员先对机构箱内气体管路进行检测。如图1。
检查发现SF6气体截止阀存在明显的泄漏点。但因为该SF6气体截止阀可能存在的气体泄漏点比较多，检修单位没有红外成像泄漏检测装置，无法准确判断泄漏位置。
检修人员经研究，采用泡沫法寻找泄漏点。经比对，发现用沐浴液兑水产生的泡沫细腻、丰富，便于涂抹。检修人员将调好的沐浴液泡沫涂在SF6气体截止阀上，发现SF6气体截止阀的确有明显的泄漏点。见图2-A，图2-B。
由于该SF6气体截止阀与开关本体灭弧室相连，不能在停电的情况下处理，而且必须进行回收气体、抽真空、注入气体处理，气候条件也必须符合要求。检修人员先补充气体到0.54MPa，运维班组对其加强了日常巡视，关注其压力变化。
2013年3月，检修人员会同厂家技术人员共同进行了缺陷消除工作，利用SF6气体回收装置回收SF6气体，更换SF6气体截止阀，更换连接处密封圈，抽真空2小时，充气至0.54MPa，经检查SF6气体不再泄漏。24小时后SF6气体微水检测为77μL/L，小于大修规定的150μL/L。运维人员经几个月的密切观察，证明该开关压力值变化正常。
为查明原因，检修人员会同厂家技术人员现场分解了SF6气体截止阀。如图3-A、图3-B、图3-C、图3-D。
图4-A所示三通密封槽内无密封圈！图4-B的盖子内也无密封圈，与图4-A所示三通用一层胶连接。此类密封容易破坏，加上断路器分合的振动，造成泄漏。
厂家人员介绍该SF6气体截止阀为外购件，没想到出厂即缺少密封圈，到底是设计问题还是生产问题也无法得知。这也是检修人员未想到的。
3.  经验体会
（1）  SF6气体带电检测是SF6断路器状态检测的必要手段。检修部门应配齐工具、设备。在此例中，由于检修部门没有SF6红外成像泄漏检测设备，被迫采用泡沫法寻找漏点，此方法无法应用于带电部位，应用受到了限制。SF6气体回收装置也是就近请厂家支援。对于断路器完全可以使用小型的SF6气体回收装置。
（2） 对于此类SF6气体截止阀，要在检修过程中避免用扳手旋转。因为该类型的SF6气体截止阀不少，其三通与盖子的密封方式并不可靠。
（3） 厂家应该对此类外购件抽样进行分解检测，避免出现此类设计或生产问题导致的家族性缺陷。
