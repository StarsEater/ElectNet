案例3. SF6泄漏成像发现500kV 断路器灭弧室SF6气体泄漏
1.案例经过
2012 年9 月25 日，XX站5624 断路器A 相SF6 压力低告警(告警值0.52MPa)，9 个小时后闭锁，当晚21 时检修人员补气至额定压力0.6MPa，9 月26 日上午11 点，现场使用红外检漏成像仪检测发现靠母线侧灭弧室下端部SF6 气体渗漏严重，同时检查表计压力已降至0.57MPa。
2.检测分析方法
现场红外检漏成像视频截图见图1。
经停电检查，发现5624 A相开关靠近交流母线侧灭弧室存在明显的SF6气体泄漏点，拆下灭弧室查看，发现瓷套法兰密封面有严重裂纹，此裂纹恰好处于产品主密封处，如图2。
此断路器为河南平高生产的产品，XX站投运以来，该类型缺陷从没发生过。经过讨论分析，认为瓷套裂纹的主要原因是瓷套在烧制加工过程中内部存在内应力，经过现场3年运行，多次操作振动后，内应力逐步释放，超出瓷套法兰面承受能力，最终出现裂纹，导致漏气。
3.经验体会
1、通过SF6 红外检漏成像仪，结合先进成熟的热成像理念及相关处理技术，利用空气和六氟化硫气体不同的红外特性，直接生成清晰的红外影像，可在远距离、不停电的情况下，快速、生成清晰的红外图像，直观的呈现泄漏点，帮助用户准确的找出断路器SF6气体泄漏点。
2、造成本次缺陷的原因是断路器灭弧室瓷套法兰密封面的裂纹。为消除此类缺陷，厂家需提升瓷套的烧制加工工艺，保证瓷套法兰面能承受操作、运行中的振动和应力。

案例5. 超高频局放及SF6分解产物测试发现220kV GIS刀闸悬浮放电
案例经过
国网成都公司公司220kV某变电站于2007年投运，2013年变电检修工区在对该站GIS设备进行超高频局放普测时发现220kV267间隔测试结果明显异常。经过多次复测并仔细对比分析，怀疑在2676刀闸气室处有放电现象。随即，工区组织人员开展SF6分解产物测试，测试结果表明2676刀闸气室SO2及H2S含量严重超标。两种带电检测结果均出现异常，综合分析初步判断该气室存在内部放电，且从超高频局放图谱判断放电类型为悬浮放电。之后，公司立即组织了力量对该间隔进行停电检查。检修人员对该气室进行开罐检查发现该气室刀闸A相传动拨叉与导电杆轴销间有明显烧蚀痕迹，内部有较多灰白色和黑褐色粉末。经分析，造成该缺陷的原因为拨叉等电位弹簧卡涉，拨叉与导电体未形成等电位，导致拨叉悬浮放电。

2.1超高频带电局部放电测试
在对该站GIS设备进行超高频局放检测时发现267间隔2676刀闸气室A相图谱存在明显异常。2676刀闸A相检测结果如图1所示。正常气室的图谱如图2所示。
对超高频图谱进行认真分析表明，2676刀闸气室A相图谱存在明显的局部放电特征，而正常气室的图谱均为噪声信号。为了保证测试结果的正确性，试验人员对该气室进行了多次复测，测试结果均表明该气室存在内部放电。复测图谱如图3所示。
对多次测试结果进行对比分析均表明该气室超高频图谱具有明显的局部放电特征，对图谱的幅值和相位特征进行进一步分析初步判断放电类型为悬浮放电。根据《电力设备带电检测技术规范》及《输变电设备状态检修试验规程》判断，具有典型局放特征的检测图谱可判断设备存在缺陷。

由表1可知该气室SO2及H2S含量严重超标，测试结果为异常。根据GB/T8905-1996《六氟化硫气体设备中气体管理和检测导则》和《四川省输变电设备状态检修试验规程实施细则》，结合超高频带电局部放电测试结果，初步确认2676刀闸气室有局部放电故障。
2.3 开罐解体检查
为了消除缺陷，工区立即组织检修人员对故障气室进行开关检查，结果发现放电点存在于2676刀闸A相闸传动拨处，见图4、图5。
由图4、图5可知，2676刀闸A相刀闸传动拨叉与导电杆轴销间有明显烧蚀痕迹，导电杆轴销间内部有较多灰白色和黑褐色粉末。显然，是因为2676刀闸A相发生局部放电导致拔叉烧灼的结果。
2.4 原因分析
经专家及经验丰富的工作人员分析，认定2676刀闸A相局部放电产生的原因在于， 2676刀闸拨叉等电位弹簧在长期工作中疲软导致卡涩，从而使得拨叉与导电杆轴销结合不紧密，与导电体未能形成稳固的等电位，导致拨叉形成悬浮放电。结合开罐检查时，发现弹簧卡涩如图6所示。
2.5 故障处理
针对这一故障情况，检修人员配合厂家技术人员更换了2676刀闸A相的传动拨叉，清理了罐体内部粉尘。同时，多次对2676刀闸气室进行气体回收、抽真空和注气工作，最大程度清洁罐体内部。在处理后试运行期间再次进行SF6分解产物测试和超高频法带电局部放电测试。分解产物测试结果如表3所示，局放测试结果如图6所示，测试结果均表明设备状态正常

案例7. SF6红外泄漏成像检测220kV主变套管气室SF6泄漏缺陷
1. 案例经过
成都供电公司220kV某变电站2号主变套管气室2010年投入运行。由于该站1号主变套管气室补气间隔小于半年，在2011年8月至10月，化学专业反复多次对套管气室进行了SF6泄漏红外成像检测。在检测过程中发现套管气室焊缝处存在SF6气体泄漏。随即检修人员根据红外检漏视频和图片进行缺陷处理。
2. 检测分析方法
该变电站套管气室在投运后，维护人员多次进行补气，泄漏情况不乐观。该套管为室内套管，离地面约为8米，无法用浓度型检漏仪进行定位。对此，检测人员只能对整个套管气室进行定点排查。
在前期的排查中，检测人员通常在白天进行检测工作，多次检测无果。经过化学专业检测人员的集体讨论，认为无法检测出的原因可能是背景干扰较大，而对于悬空的设备，无法进行近距离观察，背景的干扰影响更大。白天温度高，外界光线干扰也多，检测环境并不是很良好。
最终，检测人员决定在10月11日晚上对套管气室进行红外检测，在排查到2号主变套管B相末端时，发现焊接处有明显泄漏图像。
3. 经验体会
一、部分泄漏点有外壳遮挡，泄漏的气体通过外壳缝隙分散后浓度更低，低于检测最高灵敏度，无法检测；对于这类情况，首先采用浓度检测仪大致定位，然后进行一定的拆除工作后检测。
二、部分SF6泄漏是由于环境、运行情况造成的隐藏式泄漏，如昼夜温差、设备震动等，这是目前检漏工作中的一大难题，对于这一类的“躲猫猫”型的泄漏情况，无法通过一次红外检漏检测出泄漏点，这就需要在不同的环境下多次到现场进行检测巡视。
三，大部分泄漏点由于检修要求，需要更为严格的准确定位。然而主变套管气室等特殊位置由于高度和视角原因，不能在带电检测中近距离观察，此时不可蛮干。必要时可进行停电检测，特殊情况需要搭脚手架进行。

案例10. 超声波局放检测判定220kV GIS母线异响原因
1.案例经过
220kV某开关站的220kV GIS自2012年下半年以来，Ⅱ母桃汶线2642刀闸母线气室、Ⅱ母PT靠近228刀闸母线气室、Ⅱ母预留间隔2702刀闸母线气室（即Ⅱ母左一段气室）、Ⅲ母预留靠近2703刀闸母线气室（即Ⅲ母左一段气室）一直出现异常声响，外壳处均可感受到明显震感。设备状态评价中心在GIS Ⅲ母（1个母线气室）和Ⅱ母（3个母线气室）解体检修前后均对该GIS所有气室开展了超声波/超高频局部放电联合带电检测，超高频局放检测未发现任何放电异常。运行方对该GIS开展了SF6微水、分解产物带电检测和SF6密度巡检，也未发现异常。因此，排除了该GIS内部发生异常放电的可能。但是，Ⅱ母PT靠近228刀闸母线气室和靠近备用线2022刀闸Ⅱ母气室在超声波局放检测中均表现出幅值较高的机械振动现象。设备状态评价中心根据缺陷原因分析对GIS缺陷处理及日常巡检方面应该加强的工作提出了建议。
2.检测分析方法
（1）异常现象及带电检测工作描述
2013年2月6日，状态评价中心对该220kV变电站 GIS开展了超声波/超高频局部放电带电检测，结果如下：
Ⅱ母桃汶线2642刀闸母线气室、Ⅱ母左一段气室、Ⅲ母左一段气室这3处有异响气室的超声波局放检测结果显示：测试点信号的有效值与峰值均与变电站背景噪声水平基本相同，无50Hz和100Hz工频相关性。Ⅱ母PT靠近228刀闸母线气室超声波检测结果显示：信号的有效值和峰值比背景噪声水平稍大，有很弱的100Hz工频相关性，无50Hz工频相关性。
上述四处有异响气室的超高频局放检测结果显示：测试点信号在50个工频周期内无异常放电现象，单周期和每15分钟局部放电信号峰值和放电速率、局部放电相位图谱均无异常。
综合带电检测数据，尽管该220kV GIS母线部分气室存在异响，但是可确定所有断路器气室、隔离开关/接地开关气室、PT气室、CT气室、母线气室均无异常放电。3月，生产厂家对该站220kV GIS开展了超高频局放带电检测复测，未发现任何放电异常。运行方对该GIS开展了SF6微水、分解产物带电检测和SF6密度巡检也未发现异常。
（2）异常原因分析
在排除了GIS可能存在异常局放现象后，产生异常声音的原因有以下可能：
①外在振动引起的GIS设备薄壁件共振：如与变压器通过油气套管直接连接的GIS设备，变压器的振动有可能会引起GIS设备的机构箱、行线槽等薄壁件振动，从而产生异常声音。
②内部电场不均匀引起的局部放电声音：如某个节点产生悬浮电位、导体或外壳内壁具有尖角毛刺，当电荷积累到一定程度后，会产生连续或间断的放电声。
③电动力引起的共振：如内部主回路导体接触不良或屏蔽件松动，当电流达到一定数值时，电动力引起的振动频率与设备的固有频率接近，因共振而产生异常声音。
目前该站GIS设备与其它站变压器的连接采用架空线连接方式，附近也没有其它外在振动源，因此，可以排除外在振动引起GIS设备薄壁件共振。另外，在投运前该GIS设备已顺利通过现场工频耐压，耐压时并无异常声音，超声波/超高频局放带电检测也未查找到放电异常。因此，可基本排除内部电场不均引起局部放电声音这种可能。综上分析，该设备异响有以下两种可能：
①设备异常极有可能是由母线筒内部屏蔽件紧固不牢引起的，设备在运输中由于颠簸震动可能导致屏蔽罩出现配合间隙，由于屏蔽罩系有两颗螺钉固定，如图1示。在电动力作用下，屏蔽罩可能会绕紧固螺钉轴线方向轻微晃动而造成设备异响。
②该GIS过渡母线工字钢底架与预埋件采取自然接触的方式，没有焊接，其它位置也有底架悬空的现象。母线带电时由于电磁现象会产生微小震动，传递到壳体上的外在表现即为均匀平稳的“嗡嗡”声；特别是PT作为电磁元件，临近位置声音更大一些。当设备某外壳（或及其连接件）的固有频率与此震动频率接近时，产生共振，受激部件（即设备外壳或及其连接件）同时震动，声音明显变大。特别是过渡母线支架未与预埋基础焊接，共振未受抑制，更加剧了声音变大的外在表现。
（3）异响处理过程
3月30日起，采用了Ⅲ母、Ⅱ母轮流停电检修的方式开展消缺。检修范围涵盖Ⅲ母（1个母线气室），Ⅱ母（3个母线气室）。
①Ⅲ母停电处理情况
检修人员进入壳体内，到达异响位置，检查内部绝缘台、导体、梅花触头等的实际情况，根据相关问题，更换故障部件。
检修人员检查本气室其它触头、导体连接及螺钉紧固情况，并全部进行一次紧固处理；彻底清理壳体内部。
母线筒内检查发现三处母线屏蔽罩与触头座压接不牢，及时进行了紧固处理，再对母线壳体内部进行清洁并重新封上了母线端盖。接下来，在验证检修部位绝缘性能是否符合要求的运行电压冲击试验中，Ⅲ母左一段气室仍有异响。现场用垫片塞进工字钢与母线支架悬空处的某几个点位置，声音立刻减小。
②Ⅱ母停电处理情况

3.经验体会
①山东泰开2011年12月31日以前生产的ZF16-252GCB型GIS母线筒内屏蔽罩与触头座间均由两颗螺钉固定，该压接不牢现象在其它网省公司也出现过。2012年以后生产的该型产品由于采用了三颗螺钉固定，消除了压接不牢的现象。因此，建议关注省内在运的泰开2011年12月底前生产的ZF16-252GCB型GIS是否也有异响现象，加强巡检和带电检测力度。
②GIS设备发生异响的原因一般有三种：外在振动引起的GIS设备薄壁件共振；内部电场不均匀引起的局部放电声音；电动力引起的共振。第一种原因可根据变压器与GIS连接方式确定。第二种原因可通过超声波/超高频局放带电检测结合SF6组分分析确定。第三种原因需根据设备内部结构和安装方式确定。
③建议在GIS安装调试时将母线工字钢底架与预埋件间用焊接方式固定，而非自然接触。针对母线工字钢与预埋件重叠部分尽量采取全焊方式，而非点焊。
④充分利用GIS所在地基的沉降观测点，加强巡视，及时发现并处理由于地基沉降引起设备位移或变形的缺陷和故障发生。
