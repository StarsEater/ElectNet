设备类别：【断路器 案例 8】
案例名称：【红外成像检漏发现35kV断路器SF6泄漏缺陷】
单位名称：【国网冀北电力有限公司检修分公司】
技术类别：【红外成像检漏】 
案例经过
2013年1月30日、霸州变电站324断路器在巡视过程中发现SF6压力低，现场用红外成像检漏仪检漏，发现B相传动杆密封处存在SF6泄漏现象。
检测分析方法
作业人员将324断路器补气至额定压力后，用SF6红外成像检漏仪进行检漏，发现B相传动杆密封处存在SF6泄漏现象。SF6红外成像检漏仪与激光检漏仪不同，是利用不同物质黑体辐射不同的原理，观察SF6辐射波段，进而使SF6可视化，其好处是不需要通过高能激光触发，减少背景图像的干扰使检漏更加灵敏、准确。
经验体会
（1）因为设备漏气速率与设备气体压力成正比，故检漏前需将设备充气到额定压力，以便于发现漏点；
（2）因为设备漏气通常为间歇性漏气，故检漏中需对每个密封面观察1至2分钟，才可确认此接触面没有漏点。
