3.经验体会
当SF6气室内存在局部放电或高能放电时，SF6气体会分解成其它气体成分，以及产生一定超声振动。通过气体成分的检测和超声测试即可判断是否存在上述的缺陷。主要体会如下：
（1）SF6气体成分检测和超声波局放测试已成为辽宁省公司必须进行的带电检测项目。实践证明，通过二者的联合测试，可有效准确的发现SF6设备内部存在的放电缺陷。
（2）单一的试验方法并不能准确有效地判断设备缺陷，多种检测方法包括停电检测可有效发现设备缺陷。
（3）断路器切短路开断电流后，若吸附剂或SF6分解物较多，SF6气体分解物将超标，应及时采取更换气体或吸附剂，避免引起绝缘故障。
