电网设备状态检测技术应用案例
设备类别： 【高压开关柜】
案例名称： 【暂态地电波检测发现220kV温泉变35kV开关柜穿墙套管屏蔽环局部放电缺陷】
单位名称： 【国网浙江省电力公司金华供电公司】
技术类别： 【带电检测】
案例经过
220kV温泉变35kV开关柜为ABB UniGear type ZS 3.2铠装封闭式户内交流金属封闭开关设备，配置厦门ABB公司2006年07月生产的ZS 3.2型开关、额定电流，于2007年06月投入运行，2008年09月停电进行例行试验。
011年10月22日，在220kV温泉变对开关室内35kV高压开关柜进行局部放电检测时发现整个开关室内的TEV暂态地电位背景值非常高，达到49dB以上。通过使用PDL1的定位功能最后把放电位置定位到35kV母分开关和泉坦3614开关柜联接处，以及1#接地变开关柜与泉马3589开关柜连接处。技术人员将35kV I段母线停运排查后发现，35kV母分开关和泉坦3614开关柜联接处的母线穿墙套管、以及1#接地变开关柜与泉马3589开关柜连接处的母线穿墙套管，其套管内部与母排已有明显的放电痕迹。确定是由于设计结构及安装工艺问题，穿墙套管内部连接均压环与套管屏蔽环接触不良，开关柜内一出现潮湿现象，就容易导致套管内部与连接均压环放电。经重新对母线穿墙套管及连接均压环处理后，放电现象消失。据了解此类型的ABB UniGear type ZS 3.2开关柜穿墙套管内部放电缺陷在全国普遍存在，此后技术人员对金华电网目前应用该型号开关柜的变电站的开关柜加强了检测。
2. 检测分析方法	
2011年10月22日，在220kV温泉变35kV开关室内使用暂态地电波检测技术对35kV高压开关柜进行局部放电检测。整个开关室内的TEV背景值非常高，达到40dB以上，某些开关柜上的测试值已经达到仪器的60dB以上，通过使用PDL1的定位功能最后把放电位置定位到35kV母分开关和泉坦3614开关柜联接处，以及1#接地变开关柜与泉马3589开关柜连接处。
将开关室内的35kV I段母线停运排查后发现，35kV母分开关和泉坦3614开关柜联接处的穿墙套管、以及1#接地变开关柜与泉马3589开关柜连接处的穿墙套管，其套管内部与母排已有明显的放电痕迹。
由于设计结构及安装工艺问题，穿墙套管内部连接均压环与套管屏蔽环接触不良,又由于开关室的除湿器并未经常使用，开关柜内一旦出现潮湿现象，就容易导致套管内部放电。经重新对母线穿墙套管及连接均压环处理后，放电现象消失。此类型的ABB UniGear type ZS 3.2开关柜的放电缺陷在全国其他地方普遍存在，对此后技术人员对金华电网目前仍应用该型号开关柜的变电站的开关柜加强了检测。
3. 经验体会
a、对变电站新安装的开关柜加强验收工序，对安装工艺做好质量把关；
b、对目前仍应用该型号开关柜的变电站的开关柜加强检测，查看是否存在疑似放电，数据是否有变大变强的趋势，若需要采取相应的措施处理；同时结合有停电机会安排检查处理；       
c、对已处理的开关柜进行跟踪检测，检查处理情况是否良好。
