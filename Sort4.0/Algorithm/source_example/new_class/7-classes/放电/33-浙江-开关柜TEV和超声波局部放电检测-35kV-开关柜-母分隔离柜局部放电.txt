开关柜TEV和超声波局部放电检测发现220kV东新变电站35kV母分隔离柜局部放电
设备类别：【开关柜】
案例名称：【开关柜TEV和超声波局部放电检测发现220kV东新变电站35kV母分隔离柜局部放电】
单位名称：【国网浙江省电力公司温州供电公司】
技术类别：【带电检测】
1. 案例经过
2014年1月25日上午，试验人员对东新220kV变电站35kV开关室35kV母分隔离柜进行局放检测。TEV幅值最大28dB，超声10dB，存在局部放电现象。局放信号强烈，判断开关柜内存在严重放电，需停电检修。
2. 检测分析方法
2014年1月25日上午，试验人员对东新220kV变电站35kV开关室35kV母分隔离柜进行局放检测。TEV幅值最大28dB，超声10dB，存在局部放电现象。局放信号强烈，判断开关柜内存在严重放电，需停电检修。

3. 经验体会
（1）局部放电检测是设备状态监测的有效检测方法，能够发现电气设备绝缘内的微小放电。根据局部放电严重程度，采取适当措施，避免缺陷的进一步发展。
（2）对疑似缺陷可采用开关柜在线局放装置进行跟踪，收集不同时段的局放信号强弱，根据不同传感器采集到的局放信号时间的优先程度，大致判断局放位置。
4. 检测相关信息
检测人员：温州供电公司检修分公司变电检修室状态检测一班
检测用仪器：开关柜超声波（暂态地电压）局放检测仪
测试温度：12度，相对湿度55%
