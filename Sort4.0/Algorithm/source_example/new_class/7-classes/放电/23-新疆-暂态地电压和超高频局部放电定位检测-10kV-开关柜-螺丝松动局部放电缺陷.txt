设备类别：【高压开关柜案例(9)】
案例名称：【暂态地电压和超高频局部放电定位检测发现10kV开关柜螺丝松动局部放电缺陷】
单位名称：【国网新疆巴州供电公司】
技术类别：【带电检测】
1.案例经过
国网新疆巴州供电公司110kV某变电站10kV开关柜，型号为KYN3-10,生产日期为1997年9月1日，投运日期为1997年12月11日。2013年5月10日检测人员使用暂态地电压检测仪发现10kV 1016、1001、1022开关柜存在局部放电异常信号。14日使用超高频局部放电定位仪确定1022开关柜后柜中部断路器处存在缺陷。停电检查发现1022断路器与电流互感器连接母排的螺丝松动，处理后送电，复测无异常，缺陷消除。
2.检测分析方法
2013年5月10日检测人员使用暂态地电压检测仪发现10kV 1016、1001、1022开关柜相对值超过暂态地电压缺陷值，检测数据见表1。
2013年5月12日，5月14日对上述异常开关柜进行暂态地电压跟踪复测，发现局部放电信号仍然存在，没有明显的变化趋势，检测数据见表2。
2013年5月14日，检测人员使用超高频局部放电定位仪对1001、1016、1022开关柜进行局部放电定位（见图1），测试中将一个传感器贴在墙壁，另一个传感器贴在开关柜后柜，检测发现贴在墙壁的传感器较贴在开关柜后柜的传感器提前接受到信号（见图2）。初步断定1016、1001开关柜的放电信号来自开关室的外部，1022开关柜局部放电信号来自柜体内部。
注：红色通道信号为贴在墙壁的传感器接收的信号，黄色通道信号为贴在开关柜后柜的传感器接收的信号。
为确定1022开关柜内的局部放电信号来自哪个部位，对开关柜进行了定位测试（见图3），测试数据见图4。
根据多周期测试数据结合局部放电信号特征分析，此信号以悬浮电位放电信号为主，夹杂绝缘类型放电。经精确定位确定信号源来自10kV1022断路器开关柜的后部下方位置。
停电检查发现1022断路器与电流互感器连接排的螺丝松动，表面有明显的烧溶痕迹（见图5）。对连接排进行更换，恢复送电后，缺陷消除（见图6），暂态地电压局部放电测试结果合格，测试数据见表3。
3.经验体会
（1）暂态地电压局部放电测试仪操作简单、使用方便，能够灵敏地检测出开关柜内存在的局部放电缺陷。
（2）使用超高频局部放电定位技术可以排除变电站室外设备对开关柜局部放电测试造成的干扰，并能准确定位出局部放电缺陷的精确位置，为停电消缺指明方向。
