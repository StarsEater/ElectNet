案例二十：
设备类别： 【输电线路＋憎水性检测发现合成绝缘子憎水性丧失＋20】
案例名称： 【（硅橡胶外绝缘设备憎水性检测）发现（500kV交流架空输电线路合成绝缘子）（憎水性丧失）】
单位名称： 【（湖北省电力公司）（检修公司）】
技术类别： 【外绝缘设备憎水性检测】 
1．案例经过
国网湖北省电力公司检修公司鄂西北运维分部维护的500kV葛双一回于1982年12月投运,为提升线路绝缘水平于2007年将直线塔绝缘子更换为合成绝缘子,型号为FXBW-160，绝缘子挂网运行7年。
2013年11月对500kV葛双一回合成绝缘子进行憎水性检测时发现合成绝缘子憎水性下降严重，其中#140塔合成绝缘子憎水性为HC6（憎水性完全丧失）。
分部立即向上级主管部门汇报情况并申请技改，计划在2015年将此批合成绝缘子进行更换。
2. 检测分析方法
硅橡胶外绝缘设备憎水性检测要求及方法
（1）、 测试应选择较好的晴天进行。如迂雨雾天气，应在雨雾停止4天后进行。
（2）、喷水装置及调节
喷水装置应采用质量较好，喷出的雾状水流稳定（每次喷水量为0.7～1ml）、易调节的喷水壶。测试前，应对喷水角（喷射水流散开角为50º～70º）进行校验，校验方法如下：距喷水壶喷嘴25cm处立一张纸，垂直于该纸喷水10～15次，在纸上形成直径为25～35cm的湿斑为宜。
（3）、 测试水的要求
洁净的水。  
（4）、 喷水测试
喷水装置的喷嘴距试品25cm处，喷射方向尽可能垂直于试品表面。每秒钟喷水一次，计喷射25次，喷水后，试品表面应有水分流下。喷水结束30s内读取憎水性的分级值（HC值）。
（5）、测定结果的判定
试品表面水滴状态与憎水性的分级标准
（6）运行中复合绝缘子判定准则
（7）500kV葛双一回憎水性检测报告
3. 经验体会
随着城市经济的快速发展，部分地区环境污染日益严重，今后数年还将出现大范围、长时间雾霾天气，防污闪工作压力巨大，加强复合绝缘子的运行情况抽检工作，特别是憎水性检测，对不满足要求的设备及时治理改造，是预防电网大面积污闪事故的重要环节。
我分部在复合绝缘子憎水性检测取得的经验主要有：
（1）、 测试在的晴天进行。如迂雨雾天气，应在雨雾停止4天后进行，否则会出现较大误差。
（2）、 喷水装置对检测也有较大影响，喷水装置应采用质量较好，喷出的雾状水流稳定、易调节的喷水壶。测试前，应对喷水角进行校验，校验方法如下：距喷水壶喷嘴25cm处立一张纸，垂直于该纸喷水10～15次，在纸上形成直径为25～35cm的湿斑为宜。
（3）、 测试水的要用比较洁净的水，饮用纯净水为佳。  
（4）、喷水装置的喷嘴距试品25cm处，喷射方向尽可能垂直于试品表面。每秒钟喷水一次，计喷射25次，喷水后，试品表面应有水分流下。喷水结束30s内读取憎水性的分级值（HC值）。
