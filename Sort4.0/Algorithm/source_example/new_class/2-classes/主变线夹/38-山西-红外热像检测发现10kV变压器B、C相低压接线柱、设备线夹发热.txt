红外热像检测发现10kV变压器低压接线柱、设备线夹发热
设备类别： 【变压器案例14】
案例名称： 【红外热像检测发现10kV变压器B、C相低压接线柱、设备线夹发热】
单位名称： 【国网山西省电力公司晋城供电公司】
技术类别： 【带电检测】
1.案例经过
国网晋城供电公司检修公司配电运检专业在2014年春节前进行红外热成像检测过程中发现泽北开关站10kV826东关线东岭巷公变B相低压接线桩发热高达113.1℃，负荷电流195 A，C相设备线夹发热126℃，负荷电流338A；东岭巷公变设备参数:S9-315KVA ,生产日期：2002年04月02日，投运日期：2002年07月03日，发现缺陷后立即停电采取更换设备线夹和紧固螺栓、涂抹导电膏进行技术处理，但是进行红外热成像跟踪后，发热点依然存在，最后决定采取变压器吊芯处理，更换低压接线桩、內油珠，随后又进行红外热成像跟踪，发热点消失。
2.检测分析方法
东岭巷公变在进行红外热成像检测时，发现变压器低压侧B相低压接线桩发热温度异常，C相设备线夹发热，并对该接线桩进行了精确测温，拍下了清晰的图谱，如图1。
对图谱认真分析，结合低压侧负荷情况A相295A发现低压侧B相接线桩温度异常，B相195 A、环境温度-10℃，C相设备线夹发热，C相338A，环境温度-10℃，根据《DL/T644-2008带电设备红外诊断技术应用导则》的规定，依据电流致热型设备缺陷诊断判据，判断热像特征是以低压接线桩为中心的热像，热点明显表明，故障特征为低压侧B相接线桩与铜连接片接触不良，缺陷性质为严重缺陷，C相由于设备线夹松动，引起设备线夹发热，缺陷性质为严重缺陷，综合考虑变压器近期运行工况和低压负荷情况后，下一步进行停电对变压器进行例行试验分析。
试验结果（如表1）。
变压器低压侧B相直流电阻偏大，三相不平衡差值大47%，超出规定的4%，综合分析判断：低压接线桩与铜连接片接触不良引起发热，低压侧C相直流电阻偏大，三相不平衡差值大31%，超出规定的4%，随即制订处理缺陷方案，对历年试验数据进行比较（如表2）。
检修人员将变压器大盖打开后发现低压侧B相接线桩与铜连接片积碳严重，低压侧C相接线桩与设备线夹松动，检修人员更换低压B相低压接线桩、设备线夹、清除积碳（如图2、图3）；更换C相低压接线桩、设备线夹、紧固接线桩与铜连接片螺栓后，再次进行检测后，试验数据合格（如图4和表3）。
以上数据对红外测试结果进行了鲜明的对比，低压侧B相接线桩发热、C相设备线夹发热 进行变压器吊芯更换低压接线柱、紧固铜连接片、更换设备线夹后进行直流电阻复测，测量数据合格，在变压器恢复送电后12小时之后进行红外热成像检测跟踪，红外图谱正常（如图5）。
3.经验体会	
在实际工作应用当中、红外热成像检测技术能够行之有效的发现设备过热缺陷，为检修工作提供详细的技术参数，提高检修工作效率和质量。本专业对于配变缺陷中接头发热并发生积碳主要有以下几个方面：配变线夹与接线柱接触不紧，长时间运行导致接线柱发热。
使用线夹与配变接线柱材质不一致，发热系数不一样，长时间运行并反复热胀冷缩，导致接触电阻增大发热。
配变散热较慢，在外部发热时，会在短时间内传至内部，配变长时间外部接线柱发热后传至内部，使内部接线柱长时间发热与变压器油老化变质，慢慢形成积碳，积碳增多极易造成接地或短路。
针对以上问题，提出以下几点改进措施：
（1）配变线夹与接线柱接触处，使用上下螺母固定，并不得使用弹簧垫圈，已消除接触处接触不紧问题。
（2）要使用与配变接线柱材质一致的线夹，以防发热系数不一样，导致接触电阻增大发热。
（3）为防配变内部接线柱发热并形成积碳，除多利用红外成像技术外，也应对配变油样进行不定期抽检，以防配变内部有积碳生成。
