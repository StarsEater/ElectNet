案例三：
设备类别： 【主变压器＋在线监测系统发现500kV主变压器乙炔超标＋03】
案例名称： 【（变电站在线监测系统）发现（500kV主变压器）（乙炔超标）】
单位名称： 【（湖北省电力公司）（检修公司）】
技术类别： 【在线监测技术类别】 
1．案例经过
湖北省电力检修公司道观河500kV变电站的变电站在线监测系统[上海欣影电力科技有限公司]2013年11月5日投入运行。道500kV#2主变于2013年11月7日送电，11月8日#2主变高中压侧带负荷试运行，并经24小时运行正常。，11月10日离线油色谱分析显示C相绝缘油乙炔含量为0.71μl/L，11月15日#2主变在线油色谱装置【型号：XY-DTO】监测到C相绝缘油乙炔含量（2.48μl/L），超过告警值（1.0μl/L）。随即根据状态检修规定要求，将油色谱在线监测装置监测周期缩短为每天3次监测。监测数据多次超过告警注意值，具体情况如下表：
2013年11月19日离线油色谱检测分析显示C相绝缘油乙炔含量增长为1.13μl/L，已超过国家标准主变压器绝缘油乙炔含量标准（1μl/L），检验结果显示变压器内部存在放电现象。20日15：00—17：00省电科院设备状态评价中心对500kV道观河变电站#2主变开展超声波带电检测。经检测，#2主变C相低压侧4个检测通道采集到疑似内部放电信号，其油色谱检测结果为“油中乙炔超标且呈增长趋势”，省电科院建议对该变压器停电检查处理。20日晚上紧急停用了#2主变。
2．检测分析方法
11月21日至22日召开会议讨论检修方案。会议分析认为主变内部存在低能量的放电，此放电没有涉及到固体绝缘（CO、CO2气体正常），很可能存在于裸金属或金属联接处。本台主变厂内试验及现场安装后试验均顺利通过，所以很有可能是主变在投运合闸过程中，主变产生振动，导致某处联结处松动虚接，或有异物随油涌动到某处造成。根据现场所作的主变超声波带电检测的情况，此放电处很有可能存在于主变低压侧出线处附近。
对此确定了#2主变C相检查方案
1. 变压器放油前，测量铁芯对夹件、夹件对地、铁芯对地之间的绝缘电阻，确认是否有异常；
2. 根据现场相关规程放油；
3. 放油后，检查低压处互感器是否异常；
4. 放油后，打开低压套管安装手孔，检查低压套管尾部引线连接是否异常；
5. 检查中性点套管尾部引线连接是否异常
6. 打开开关侧人孔，进行通风，待油箱内氧气含量达到规定值、并作好安全防范措施之后，进行钻芯检查；
检查部位包括：
6.1 两侧油箱磁屏蔽是否有异常；
6.2 铁芯旁轭与上下铁轭连接处是否有异常（伸出的尖角是否有短路、磕碰）；D、F区
6.3 下铁轭上表面是否有金属异物；C区
6.4 铁芯旁轭屏蔽接地是否良好；B区
6.5 铁芯下铁轭屏蔽接地是否良好；
6.6 夹件屏蔽线接地是否良好；A区
6.7 上下侧梁连接处连接是否良好；E区
6.8 其它可以检查到的部位是否有异常；
11月24日下午主变检修开工，初步检查35KV侧直阻及绝缘检查正常，上午11点检查35kv侧内部未发现明显放电痕迹，25日12点厂家人员进入#2变压器C相内部检查，发现C相本体铁芯有多处硅钢片之间的短路现象，现场进行了处理，经过处理前后照片比对，电科院、检修公司、中超公司、省送变电公司、厂家人员共同研究决定，处理情况正常，进入下一道抽真空工序。11月30日12时30分热油循环完毕，进入静置72小时阶段，12月3日13时静置完毕，省送变电公司取油样，乙炔含量为0.04PPM，12月4日13点省电科院到站进行C相带电局放和变形测量，静置24小时后，5日省送变电公司取油样，乙炔含量为0.13PPM；6日省检修公司变电检修中心到站取油样乙炔含量为0.21PPM。9日，由中超公司、省基建部、省送变电公司、省检修公司、省电科院在道观河站开#2主变送电协调会，确定了#2主变的送电方案。10日#2主变送电前，省送变电公司取油样乙炔含量为0.10PPM。10日20:30道观河#2主变送电，由高压侧道5022、5023开关带低压侧道61开关运行，中压侧道231开关单元为冷备用。11日下午3点，省检修公司变电检修中心到站取油样乙炔含量为0.43PPM。12日上午省送变电公司取油样，乙炔含量为0.35PPM。12日晚上主变中压侧带电，跟踪检测数据如下表：

3．经验体会
大量事实说明变电站综合监测系统对于变压器油色谱监测具有良好的灵敏性，其数据稳定性跟环境及装置工作方式有一定关系，但在线监测结果和离线分析数据一起配合能够有效指导设备状态检修，提高检修工作效率和质量。
